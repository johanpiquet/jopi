# Jopi

`jopi` is a CLI tool for Jopi.

In current versions, it allows bootstrapping a project through templates.
In future versions, it will allow some admin tasks around modules, routes, etc..

To know:
* The env var `JOPI_INIT_USE_DEV_DIR` allows pointing to a local directory to clone templates from.
